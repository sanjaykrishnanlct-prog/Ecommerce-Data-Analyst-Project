# Ecommerce-Data-Analyst-Project
End-to-End E-Commerce Sales Analytics Project using Python, Pandas, NumPy, Matplotlib, Data Cleaning, EDA, Visualization, and Business Insights.

## Business Problem

Analyze sales performance, customer behavior,
payment trends and revenue growth.

## Dataset

1. Orders Dataset
2. Customers Dataset
3. Payments Dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Workflow

1. Data Loading
2. Data Cleaning
3. Data Transformation
4. Data Merging
5. Exploratory Data Analysis
6. Data Visualization
7. Business Insights
8. Recommendations

## Key Insights

- Tamil Nadu generated highest revenue.
- Phone category generated highest revenue.
- UPI is the most used payment method.
